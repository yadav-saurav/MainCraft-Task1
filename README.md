# MainCraft-Task1

📌 Project Overview
This repository contains the completion of Task 1 for the Full-Stack Web Development Internship at Maincrafts. The objective of this task was to build a clean, responsive, and modern landing page landing interface utilizing semantic HTML5, modern CSS3 layout paradigms (Flexbox and CSS Grid), and global style configurations.

🚀 Features Implemented
Header & Navigation: Includes a distinct brand logo, standard primary navigation flows (Home, Features, Services, Contact), and a functional CSS-driven hover dropdown menu nested under Services.
Hero Section: An immersive landing area featuring a bold heading, contextual sub-text tagline, background overlay blending, and an actionable Call-To-Action (CTA) button.
Features Grid Section: A fully modular 4-card grid layout highlighting key performance traits (Blazing Fast, Fully Responsive, Highly Scalable, and an added Secure Architecture layer) utilizing FontAwesome visual indicators.
Semantic Footer: Contains the dynamic copyright marker along with layout breaks for Privacy Policy, Terms, and Contact parameters.
🎨 Design & Modernization Details
Typography: Dual-font pairing architecture curated from Google Fonts featuring Plus Jakarta Sans for tracking prominent headlines and Inter for rendering body copy legibility.
Color Scheme: Deep slate primary hues #0f172a paired with fluid interactive electric blue highlights #3b82f6 for modern software/tech layout guidelines.
Responsiveness: Fluid breakpoints configured to handle layout restructures across viewport widths—collapsing to unified vertical stacks on smaller smartphone interfaces.
🛠️ Tech Stack Used
HTML5 (Semantic structures)
CSS3 (Custom Variables, Flexbox, CSS Grid, Transitions)
FontAwesome Icons v6.0 (CDN integration)
Google Fonts API
